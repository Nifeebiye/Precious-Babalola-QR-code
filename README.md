## Overview
This project is a simple QR code card built using HTML and CSS. The goal of the project was to practice fundamental front-end development concepts such as layout structuring, styling and element positioning.
The design consists of a centered card placed on a soft background, containing a QR code image and descriptive text. I used a flexbox to achieve proper alignment and centering, ensuring the layout remains clean and visually balanced across different screen sizes.

Through this project, I strengthened my understanding of structuring web pages with semantic HTML, styling with CSS, using flexbox for layout and alignment, and applying spacing, padding, and visual hierarchy for better UI design.

## My process
I started by analyzing the design and breaking it down into simple sections: the background, the card container, and the content within the card (QR image and text). This helped me understand the structure before writing any code.
Next, I built the basic HTML structure, focusing on organizing the elements clearly and logically.
For styling, I began with the overall layout by centering the card on the page using Flexbox. Once the layout was in place, I moved on to styling the card itself by adding padding, border-radius, and spacing to achieve a clean and modern look.
After that, I styled the content inside the card, ensuring the image fit properly and the text was well-aligned and readable.
Finally, I refined the design by adjusting spacing, alignment, and minor visual details to closely match the original design while maintaining simplicity and responsiveness.
